# StudyHub - 智能學習平台

一個現代化的學習社交平台，連接全球學習者。

## 功能特色

- 👥 智能好友系統
- 💬 即時聊天功能
- 🌐 多語言支援
- 📚 學習資源共享
- 🎯 個人化學習分析
- 🛡️ 管理員管理面板

## 技術架構

### 前端
- HTML5 + CSS3 + JavaScript
- 響應式設計
- 現代化 UI/UX

### 後端
- Python + FastAPI
- SQLite 資料庫
- JWT 身份驗證
